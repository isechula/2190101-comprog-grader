### ​Gymnastic ​Score (​★)

[Instructions](https://2190101.nattee.net/problems/853/get_statement/03_If_03.pdf)\
Solutions:

```python
scores = [float(i) for i in input().split()]

#Im limiting myself to only using things teacher has taught
largest = scores[0]
if scores[1] > largest:
    largest = scores[1]
if scores[2] > largest:
    largest = scores[2]
if scores[3] > largest:
    largest = scores[3]

smallest = scores[0]
if scores[1] < smallest:
    smallest = scores[1]
if scores[2] < smallest:
    smallest = scores[2]
if scores[3] < smallest:
    smallest = scores[3]
    
print(round((sum(scores) - largest - smallest)/2,2))
```

```python
#I dub this the "bubblesort solution"
a,b,c,d = [float(i) for i in input().split()]

#basically bubblesort the largest number to the top
if a > b:
    a,b = b,a
if b > c:
    b,c = c,b
if c > d:
    c,d = d,c
#same thing backwards
if c < b:
    c,b = b,c
if b < a:
    b,a = a,b
    
print(round((b+c)/2,2))
```

### ​DayOfYear (​★★)

[Instructions](https://2190101.nattee.net/problems/858/get_statement/03_If_08.pdf)\
Solutions:

```python
#Im limiting myself to only using things teacher has taught
days_in_month = [31,28,31,30,31,30,31,31,30,31,30,31]

day,month,year = [int(input()) for i in range(3)]
year -= 543

days = sum(days_in_month[:month-1]) + day

is_leap_year = year % 400 == 0 or (year % 4 == 0 and year % 100 != 0)
if month > 2 and is_leap_year:
    days += 1

print(days)
```
